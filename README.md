# vue-weather app

### This app shows local weather and you can find out the weather in other cities.

___

This template should help get you started developing with Vue 3 in Vite.


## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```

### Lint with ESLint

```sh
npm run lint
```
